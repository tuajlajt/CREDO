## Clinical Responsibility

This agent handles drug interaction checking with severity flagging.
All outputs must:
1. Be reviewed by a qualified pharmacist or physician before clinical action
2. Include the data source and interaction database version
3. Include severity classifications per standard scales (major/moderate/minor)
4. Never override clinical judgement

---

## Agent Workflow (at inference time)

```
Input: list of drug names (as prescribed, trade names or generics)
        ↓
Step 1: Map each drug name → active ingredient(s)
        ↓
Step 2: Get all pairs of active ingredients
        ↓
Step 3: Check each pair against interaction database
        ↓
Step 4: Classify severity of each interaction found
        ↓
Step 5: For MAJOR severity: generate alternative combinations
        ↓
Output: structured interaction report with recommendations
```

## Implementation

```python
# src/agents/pharmacology_agent.py
from dataclasses import dataclass
from itertools import combinations
from typing import Optional
import requests

@dataclass
class Drug:
    name: str                          # As prescribed
    active_ingredients: list[str]      # Mapped from database
    dose: Optional[str] = None
    route: Optional[str] = None

@dataclass
class Interaction:
    drug_a: str
    drug_b: str
    severity: str                      # "major" | "moderate" | "minor"
    description: str
    mechanism: Optional[str] = None
    clinical_significance: Optional[str] = None
    source: str = ""

@dataclass
class PharmacologyReport:
    drugs_checked: list[Drug]
    interactions: list[Interaction]
    major_interactions: list[Interaction]
    recommendations: list[str]
    alternatives: dict[str, list[str]]  # drug -> alternative options
    disclaimer: str
    database_version: str
    requires_review: bool = True


class PharmacologyAgent:
    """
    Drug interaction checker.
    Checks interactions among active ingredients of a medication list.
    """

    DISCLAIMER = (
        "This interaction check is generated by AI and is not a substitute "
        "for clinical pharmacist review. Always consult a qualified healthcare "
        "professional before making medication changes."
    )

    def __init__(self, config: dict):
        self.config = config
        self.drug_db = DrugDatabase(config["drug_database_path"])
        self.interaction_db = InteractionDatabase(config["interaction_database"])

    def check_interactions(self, drug_names: list[str]) -> PharmacologyReport:
        """
        Main entry point. Takes a list of drug names, returns full report.
        """
        # Step 1: Map to active ingredients
        drugs = [self._map_to_active_ingredients(name) for name in drug_names]

        # Step 2: Get all active ingredient pairs
        all_ingredients = []
        for drug in drugs:
            all_ingredients.extend(drug.active_ingredients)
        ingredient_pairs = list(combinations(set(all_ingredients), 2))

        # Step 3: Check each pair
        interactions = []
        for ing_a, ing_b in ingredient_pairs:
            found = self.interaction_db.check(ing_a, ing_b)
            interactions.extend(found)

        # Step 4: Classify and separate by severity
        major = [i for i in interactions if i.severity == "major"]
        moderate = [i for i in interactions if i.severity == "moderate"]

        # Step 5: Generate alternatives for major interactions
        alternatives = {}
        recommendations = []
        for interaction in major:
            alts = self._find_alternatives(interaction, drugs)
            if alts:
                alternatives[interaction.drug_a] = alts
                recommendations.append(
                    f"MAJOR interaction between {interaction.drug_a} and "
                    f"{interaction.drug_b}. Consider: {', '.join(alts)}"
                )

        if not interactions:
            recommendations.append("No significant interactions identified in database.")

        recommendations.append(self.DISCLAIMER)

        return PharmacologyReport(
            drugs_checked=drugs,
            interactions=interactions,
            major_interactions=major,
            recommendations=recommendations,
            alternatives=alternatives,
            disclaimer=self.DISCLAIMER,
            database_version=self.interaction_db.version,
            requires_review=True,
        )

    def _map_to_active_ingredients(self, drug_name: str) -> Drug:
        """
        Map trade name or generic name → active ingredient(s).
        Uses local drug database + fallback to RxNorm API.
        """
        ingredients = self.drug_db.get_active_ingredients(drug_name)

        if not ingredients:
            # Fallback: RxNorm API
            ingredients = self._rxnorm_lookup(drug_name)

        if not ingredients:
            # Unknown drug — flag it, don't silently skip
            ingredients = [drug_name]
            import logging
            logging.warning(f"Could not map '{drug_name}' to active ingredient. "
                           f"Using drug name directly.")

        return Drug(name=drug_name, active_ingredients=ingredients)

    def _rxnorm_lookup(self, drug_name: str) -> list[str]:
        """Look up active ingredients via NCBI RxNorm API."""
        try:
            url = f"https://rxnav.nlm.nih.gov/REST/rxcui.json?name={drug_name}&search=1"
            r = requests.get(url, timeout=5)
            rxcui = r.json()["idGroup"]["rxnormId"][0]

            url2 = f"https://rxnav.nlm.nih.gov/REST/rxcui/{rxcui}/related.json?tty=IN"
            r2 = requests.get(url2, timeout=5)
            concepts = r2.json()["relatedGroup"]["conceptGroup"]
            for group in concepts:
                if group.get("tty") == "IN":
                    return [c["name"] for c in group.get("conceptProperties", [])]
        except Exception:
            return []
        return []

    def _find_alternatives(self, interaction: Interaction,
                            current_drugs: list[Drug]) -> list[str]:
        """
        Suggest alternative drugs in the same therapeutic class
        that do not interact with the other drug in the pair.
        Consults drug class database + TxGemma if available.
        """
        therapeutic_class = self.drug_db.get_therapeutic_class(interaction.drug_a)
        if not therapeutic_class:
            return []

        alternatives = self.drug_db.get_class_alternatives(
            therapeutic_class, exclude=interaction.drug_a
        )

        # Filter: keep only alternatives that don't interact with drug_b
        safe_alternatives = [
            alt for alt in alternatives
            if not self.interaction_db.check(alt, interaction.drug_b)
        ]

        return safe_alternatives[:3]  # Return top 3
```

## Interaction Databases

Recommended sources (check licensing for your use case):

| Database | Coverage | License | API Available |
|---|---|---|---|
| DrugBank | 14k+ drugs, 2M+ interactions | Commercial / Research | Yes |
| KEGG DRUG | Drug-drug interactions | Academic | Yes |
| RxNorm (NCBI) | Drug name mapping | Free (US gov) | Yes |
| SIDER | Side effects | Creative Commons | Download |
| OpenFDA | Adverse events | Free | Yes |

For production: DrugBank or a validated clinical interaction database.
For development/testing: OpenFDA + RxNorm (both free).

## Severity Classification

Use standard clinical severity levels:

- **Major**: Potentially life-threatening or capable of causing permanent damage.
  Action required — alert prescriber immediately.
- **Moderate**: May cause clinical deterioration; require monitoring or adjustment.
  Highlight for clinical review.
- **Minor**: Minimally clinically significant; monitoring or dosage adjustments unlikely needed.
  Log but do not alert.

## Red Flags

- Removing `requires_review: true` from any output
- Silently skipping a drug that couldn't be mapped to active ingredient
- Using an unvalidated or outdated interaction database
- Presenting major interaction alert without recommendation
- Not logging database version with every interaction check
